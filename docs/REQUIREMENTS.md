# 项目需求文档：工作学习计划表（网页端）

## 一、计划表核心功能

### 1.1 计划表展示

#### 1.1.1 时间轴
- Y 轴：固定显示每日时间轴，范围 06:00–23:00。
- 刻度与网格：以小时为单位进行网格划分，支持半小时刻度显示。
#### 1.1.1 时间轴（补充）
- X 轴：支持切换查看周期，包括但不限于 3 天、7 天、15 天、自定义日期范围。

#### 1.1.2 视图切换
- 提供日视图、周视图、月视图（可选，主要展示任务分布而非详细时间轴）。

### 1.2 任务管理

#### 1.2.1 任务创建
- 在计划表空白处双击或点击 + 按钮，快速创建新任务。
- 展示：任务默认显示为圆角矩形方块。

#### 1.2.2 任务编辑
- 重命名：双击任务方块，直接在方块内编辑任务名称。

#### 1.2.3 任务操作
- 拖拽：任务方块可在 Y 轴时间线和 X 轴日期线上自由拖拽移动。
- 复制：按住特定键（如 Alt/Ctrl）拖拽任务，实现任务的快速复制。
- 删除：选中任务后，点击删除按钮或按 Delete 键删除任务。
- 高亮显示：单击任务方块，该任务被高亮显示，便于识别和操作。

#### 1.2.4 任务属性（可选）
- 任务分类/标签：为任务添加学习、工作、生活等标签，可自定义标签颜色。
- 任务优先级：设置高中低优先级，通过不同颜色或图标区分。
- 任务备注：为任务添加详细说明或备注信息。

#### 1.2.5 时间调整
- 拖拽任务方块的上下边缘，调整任务的开始时间和结束时间。

#### 1.2.6 日期调整
- 拖拽任务方块，将其移动到不同日期。
#### 1.2.7 任务完成状态
- 标记任务为已完成或未完成。
- 已完成任务采用不同样式显示（如文字删除线、颜色变浅/降低不透明度）。
### 1.3 导出

#### 1.3.1 导出为图片
- 用户可选择当前视图（如 3 天、7 天）或指定日期范围的计划表，导出为 PNG/JPG 图片文件。
- 导出图片需保证高分辨率与清晰度，便于打印或在移动端查看。

#### 1.3.2 主题设置
- 提供多种界面主题（亮色/暗色模式），导出图片应与当前主题一致。
